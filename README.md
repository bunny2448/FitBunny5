# FitBunny - PWA Workout Manager

## How to Deploy to your iPhone

1. **Sync to GitHub**: Push this code to your GitHub repository.
2. **Enable Pages**:
   - Go to Repo **Settings** > **Pages**.
   - Under **Source**, select **GitHub Actions**.
3. **Install**:
   - Wait about 1-2 minutes for the "Deploy" action to finish (check the **Actions** tab).
   - Visit the URL GitHub gives you on your iPhone.
   - Tap Share > Add to Home Screen.
